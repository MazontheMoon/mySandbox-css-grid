First things first - Create the grid

A grid is created by specifying the number of rows and columns.

This example is for a 3 column x 2 row grid

  • Method 1 - using set measurements

            .container {
          display: grid;
          grid-template-columns: 10% auto 100px;
          grid-template-rows: 50px 50px;
          grid-gap: 3px;
        }

    Note: minmax can also be used for when columns/rows resize

            .container {
          display: grid;
          grid-template-columns: 100px minmax(100px, 500px) 100px;
          grid-template-rows: 50px 50px;
          grid-gap: 3px;
        }

  • Method 2 - using fractions

            .container {
          display: grid;
          grid-template-columns: 1fr 1fr 1fr;
          grid-template-rows: 50px 50px;
          grid-gap: 3px;
        }

  • Method 3 - using repeat

            .container {
          display: grid;
          grid-template-columns: 50px repeat (3, 1fr);
          grid-template-rows: repeat (2, 1fr)
          grid-gap: 3px;
        }

  • Method 4 - auto rows

            .container {
          display: grid;
          grid-template-columns: 50px repeat (3, 1fr);
          grid-auto-rows: 100px;
          grid-gap: 3px;
        }
        
            .container {
          display: grid;
          grid-template-columns: 50px repeat (3, 1fr);
          grid-auto-rows: minmax(100px, auto);
          grid-gap: 3px;
        }

  • Method 5 - grid template

    grid-template is shorthand for grid-template-rows and grid-template-columns(rows/columns)

            .container {
          display: grid;
          grid-template: repeat(2, 50px) / repeat(3, 1fr); 
          grid-gap: 3px;
        }

  • Method 6 - auto-fill

    For this example the grid will create as many 200px columns that will fit in container 

            .container {
          display: grid;
          grid-template-columns: repeat(auto-fill, 200px);  
          grid-gap: 3px;
        }

    For this example the grid will create as many 200px columns that will fit in container with remaining space shared 

            .container {
          display: grid;
          grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));  
          grid-gap: 3px;
        }

    Auto-fill will add extra columns with no content – shows as blank space

  • Method 7 - auto-fit

    For this example the grid will create as many 200px columns that will fit in container 

            .container {
          display: grid;
          grid-template-columns: (repeat(auto-fit, minmax (100px), 1fr);  
          grid-gap: 3px;
        }

  • Method 8 - naming grid lines

    Yet another method of doing the same thing…for larger projects easier than going by row start numbers, etc (also main-start and menu-start could both be assigned to same grid area. 

            .container {
          display: grid;
          grid-template-columns: [main-start] 1fr [content-start] 1fr [content-end] 1fr [main-end];
          grid-template-rows: [main-start] 100px [content-start] 100px [content-end] 100px [main-end];
          grid-gap: 3px;
        }

    Elements are then positioned by defining the row/column start/end

                  .item {
                grid-column-start: main-start;
                grid-row-start: main-start;
                grid-row-end: main-end
              }

Second things secondly - Content in the grid

Items are aligned within their assigned grid area

  • Self alignment

            .item {
          align-self: stretch;
          justify-self: center;
        }

    Properties include:

    • auto
    • normal
    • start
    • end
    • center
    • stretch
    • baseline
    • first baseline
    • last baseline

  • Container alignment

    Items are aligned on grid tracks within their assigned grid area

            .item {
          align-self: stretch;
          justify-self: center;
        }

    Properties include:

    • normal
    • start
    • end
    • center
    • stretch
    • space-around
    • space-between
    • space-evenly
    • baseline
    • first baseline
    • last baseline

align-items (container) vs align-self (item) and justify-items (container) vs justify-self (item)

An example of laying out article using grid

    article {
      display: grid;
      grid-template-columns: 80px 1fr 80px;
      }

    article > * {
    grid-column: 2; //start at line 2 – gives border effect
    min-width: 0; //if this is not set it looks off
    }

    article > figure {
      grid-column: 1 / -1; //this takes whole width
      margin: 20px 0; //using full row so need a bit of a margin
    }

    article> .aside {
      grid-column: 3; //setting it in the last column looks like an aside/side note
      color: #666; //distinguish aside
      font-size: 0.8em; //smaller text
   }
